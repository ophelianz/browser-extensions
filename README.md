# browser extensions for ophelia

---

build from source

- chrome

```bash
git clone https://github.com/ophelianz/browser-extensions.git
cd browser-extensions/chrome
bun dev
bun run build
```

- firefox

```bash
git clone https://github.com/ophelianz/browser-extensions.git
cd browser-extensions/firefox
bun dev
bun run build
```
